# 무한 스크롤 (Infinite Scroll)

## 개요

사용자가 페이지의 특정 지점에 도달하거나 버튼을 클릭할 때 더 많은 데이터를 로드하는 기능.
모든 데이터를 한 번에 가져오는 것보다 효율적이며, 특히 데이터가 많은 SNS 타임라인 같은 서비스에 적합하다.

## 구현 방식

- 사용자가 새 데이터를 요청하는 **버튼 클릭** 시 fetch
- 사용자가 페이지 **특정 지점으로 스크롤** 시 fetch (본 강의에서 사용하는 방식)

## 페이지네이션 vs 무한 스크롤

|               | 페이지네이션                                    | 무한 스크롤                              |
| ------------- | ----------------------------------------------- | ---------------------------------------- |
| 페이지 추적   | 컴포넌트 state                                  | useInfiniteQuery가 자동 추적             |
| 페이지 전환   | 버튼 클릭 → state 업데이트 → query key 업데이트 | 스크롤 위치 감지 → 자동 fetch            |
| API 응답 형식 | 단순 배열                                       | 객체 (results + next + previous + count) |

## useInfiniteQuery

무한 스크롤 구현을 위한 TanStack Query 훅.

```ts
// Star Wars API 응답 형식
{
  results: [],   // 데이터 배열
  next: "URL",   // 다음 페이지 URL
  previous: null, // 이전 페이지 URL (첫 페이지면 null)
  count: 82      // 전체 데이터 수
}
```

- `next`, `previous` 값을 통해 React Query가 다음 쿼리를 자동으로 추적
- 하나의 쿼리 키에 계속 데이터가 추가되는 방식으로 동작
- 사용자가 하단에 가까워지면 미리 fetch하여 **중단 없는 스크롤** 제공

# useInfiniteQuery 프로젝트 세팅

## 초기 세팅 순서

### 1. 의존성 설치

```bash
# react query + devtools 한 번에 설치
npm install @tanstack/react-query @tanstack/react-query-devtools
```

### 2. QueryClient 생성 및 Provider 설정 (App.jsx)

```tsx
import { QueryClient, QueryClientProvider } from "@tanstack/react-query";
import { ReactQueryDevtools } from "@tanstack/react-query-devtools";

const queryClient = new QueryClient();

function App() {
  return (
    <QueryClientProvider client={queryClient}>
      {/* 컴포넌트들 */}
      <ReactQueryDevtools />
    </QueryClientProvider>
  );
}
```

> DevTools는 별도 라이브러리로 분리되어 있음 (버전 관리 및 빌드 최적화 목적)
> `NODE_ENV=development` 일 때만 렌더링됨 → `npm run dev` 시에만 표시, 빌드 후엔 미표시

## 프로젝트 구조

```
src/
├── people/
│   ├── InfinitePeople.jsx   # 무한 스크롤 - 인물 목록
│   └── Person.jsx           # 인물 컴포넌트 (name, hair_color, eye_color)
└── species/
    ├── InfiniteSpecies.jsx  # 무한 스크롤 - 종족 목록
    └── Species.jsx          # 종족 컴포넌트 (name, language, average_lifespan)
```

## API

- **People** : `https://swapi.dev/api/people/`
- **Species** : `https://swapi.dev/api/species/`
- 공통 유틸 함수 `fetchUrl(url)` 사용
