# InfiniteScroll 컴포넌트 적용

## react-infinite-scroller

`useInfiniteQuery`와 함께 쓰기 좋은 무한 스크롤 라이브러리.
스크롤 위치 감지를 알아서 해주고, 필요한 시점에 `loadMore` 함수를 자동 호출해줌.

---

## 핵심 props

| props      | 연결할 값       | 설명                     |
| ---------- | --------------- | ------------------------ |
| `loadMore` | `fetchNextPage` | 더 불러올 때 실행할 함수 |
| `hasMore`  | `hasNextPage`   | 다음 데이터 존재 여부    |

---

## 구현 코드

```jsx
const { data, fetchNextPage, hasNextPage, isFetching } = useInfiniteQuery({
  queryKey: ["sw-people"],
  queryFn: ({ pageParam = initialUrl }) => fetchUrl(pageParam),
  getNextPageParam: (lastPage) => lastPage.next || undefined,
});

return (
  <InfiniteScroll
    loadMore={() => {
      if (!isFetching) fetchNextPage(); // 이미 fetching 중이면 중복 호출 방지
    }}
    hasMore={hasNextPage}
  >
    {data.pages.map((pageData) =>
      pageData.results.map((person) => (
        <Person
          key={person.name}
          name={person.name}
          hairColor={person.hair_color}
          eyeColor={person.eye_color}
        />
      )),
    )}
  </InfiniteScroll>
);
```

## data.pages 구조

```js
data.pages = [
  // 1페이지 fetch 결과
  { results: [{ name: "Luke", hair_color: "blond", eye_color: "blue" }, ...], next: "...page=2" },
  // 2페이지 fetch 결과
  { results: [{ name: "C-3PO", hair_color: "n/a", eye_color: "yellow" }, ...], next: "...page=3" },
  ...
]
```

그래서 map을 2번 써야 함

```js
data.pages.map((pageData) =>       // 페이지 단위로 순회
  pageData.results.map((person) => // 각 페이지의 실제 데이터 순회
    <Person ... />
  )
)
```

## 주의사항

### isFetching으로 중복 호출 방지

```js
loadMore={() => {
  if (!isFetching) fetchNextPage()  // fetching 중이면 스킵
}}
```

### 에러/로딩 처리 필요

```
data가 undefined인 상태에서 data.pages 접근하면 에러 발생
→ isLoading, isError 처리 필수!
```

```jsx
if (isLoading) return <div>로딩 중...</div>;
if (isError) return <div>에러 발생!</div>;
```
